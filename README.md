# Curriculum_management
Curriculum Management System
Overview

The Curriculum Management System is a full-stack web application designed to manage academic curriculum structures efficiently. The platform allows users to create, organize, and manage Programs, Courses, Subjects, Modules, Topics, and Learning Objectives through an interactive drag-and-drop interface.

The system helps educational institutions simplify curriculum planning, improve content organization, and maintain structured academic workflows.

Features
Drag-and-drop curriculum builder
Hierarchical curriculum structure management
Program → Course → Subject → Module → Topic organization
Dynamic curriculum editing and reordering
File upload support
Dockerized deployment setup
Responsive frontend UI
REST API integration
Backend data management
Cloudinary integration for file storage
Modern full-stack architecture
